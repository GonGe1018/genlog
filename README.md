# 🌱 GENLOG - 생성형 블로그 서비스
### -2024 선린인터넷고등학교 서버 2학기 수행평가 프로젝트-
주제나 키워드만 입력하면 LLM을 사용하여 자동으로 블로그 글을 생성하는 서비스<br><br>

## 🚀 설치
```bash
git clone https://github.com/GonGe1018/genlog.git
cd genlog

pip install -r requirements.txt

cd frontend
yarn install
```

## 🏄 사용법
```
python -m manage.py runserver
cd frontend
yarn run start
```
